16x Media Digest

A free Mac utility that turns Urdu and English audio or video into a clean, structured English Markdown digest — entirely on your machine. Local Whisper transcription, smart context-window chunking, and your choice of a local LLM (Ollama) or a cheap cloud fallback (OpenRouter).

Get started How it works
macOS (Apple Silicon) Tauri + React whisper.cpp Ollama / OpenRouter اردو + English

How it works

The app orchestrates the whole pipeline deterministically. The LLM only ever sees one small, well-scoped task at a time — which is why cheap 7–9B models produce reliable digests.

01
Drop media
Drag in any .mp4, .mov, .mp3, .m4a, .wav — lectures, podcasts, meetings, vlogs.
02
Extract audio
FFmpeg converts to 16 kHz mono WAV — the exact format Whisper needs.
03
Transcribe locally
whisper.cpp runs on-device with auto Urdu/English detection (or manual toggle), keeping timestamps.
04
Smart chunking
Transcript splits on sentence boundaries (both ۔ and .) into ~500-word chunks that never overflow a small model's context.
05
Digest
Each chunk is translated (if Urdu) and summarized, then stitched into one timestamped Markdown document. Copy with one click.

Private by default

With Ollama selected, nothing leaves your Mac — media, transcript, and digest all stay local. OpenRouter is an explicit opt-in fallback.

Built for bilingual reality

Handles pure Urdu, pure English, and the code-switched mix real conversations actually are. Output is always clean English Markdown.

Small-model friendly

Deterministic chunking keeps every LLM request under ~2.2k tokens, so 4k-context models like Gemma 2 9B work reliably.

Setup guide

1.Install the app

Download the latest .dmg from the link above (coming soon) or build from source:

git clone https://github.com/16xbrains/media-digest
cd media-digest && npm install && npm run tauri build

2.Install FFmpeg & whisper.cpp

brew install ffmpeg
brew install whisper-cpp

Then download a multilingual Whisper model. small is a good speed/accuracy balance for Urdu; use medium for noisy audio:

curl -L -o ~/whisper-models/ggml-small.bin \
  https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-small.bin

Point the app to this model file in Settings → Transcription. Avoid .en models — they are English-only.

3.Choose your LLM provider

Option A — Ollama (local, free, private)
brew install ollama
ollama serve
ollama pull gemma2:9b   # or qwen2.5:7b

In the app: Settings → Provider → Ollama. Default endpoint http://localhost:11434 is pre-filled. qwen2.5:7b has notably strong Urdu.

Option B — OpenRouter (cloud fallback, ~pennies/hour of audio)
  1. Create a key at openrouter.ai/keys
  2. Settings → Provider → OpenRouter → paste key
  3. Pick a cheap model, e.g. google/gemini-flash-1.5

Only transcript text is sent — never your media files. The key is stored in the macOS Keychain.

Using the app

1
Drop a file. Drag any audio/video onto the window. The status bar walks through each stage: Extracting audio → Transcribing → Processing chunk 3/7 → Done. The UI never freezes — everything heavy runs in background processes.
2
Set the language (optional). Auto-detect works well; force اردو or English from the toggle if a heavy accent or code-switching confuses detection.
3
Review the digest. Each section is headed by its media timestamp range (e.g. ## [12:40–15:05]) so you can jump back to the source. Hit Copy to Clipboard and paste into Notion, Obsidian, or a doc.
4
Rerun cheaply. Transcripts are cached — switching provider or model re-digests without re-transcribing.

FAQ

Does my audio ever leave my Mac?

Never. Transcription is always local (whisper.cpp). With Ollama, the digest step is local too. Only if you opt into OpenRouter is transcript text sent to the cloud.

Why chunk the transcript instead of sending it all at once?

Small local models have 4k–8k token context windows and degrade badly when overloaded. The app splits transcripts on sentence boundaries into ~500-word chunks with one sentence of overlap, so every request stays small and every summary stays sharp.

How fast is it?

On an M-series Mac with the small Whisper model, transcription runs several times faster than real time. A 1-hour Urdu lecture typically digests end-to-end in 10–15 minutes fully locally.

Which languages are supported?

Urdu and English (including mixed/code-switched speech) are first-class. Whisper itself is multilingual, so other languages often work — but they're untested.

What does it cost?

The tool is free. Fully local usage costs nothing. Via OpenRouter, a 1-hour recording is typically under $0.02 with a flash-class model.
